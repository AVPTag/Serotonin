# Serotonin
Serotonin is a semi-tethered/semi-untethered jailbreak for iOS 17-18.7.2 targetting A4-A18 utilizing the Safari WebKit Leftover Bugs.

|| NOTE: Serotonin is no longer based off of Dopamine. ||

Serotonin is a semi tethered / semi untethered jailbreak for iOS and iPadOS (soon to cover more) 17-18.7.2. The jailbreak will target all devices running A4-A11, A12-A18, and M1-M2.
